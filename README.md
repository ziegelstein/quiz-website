# Quiz Website

A small self-contained project to create a quiz website using HTML, CSS, and JavaScript. The website displayes questions for a quizmaster to ask to the contestants. The quizmaster can click on the question to reveal the answer.

The questions are loaded from a JSON file, which allows for easy updating and maintenance of the quiz content. The website is designed to be simple and user-friendly, making it easy for the quizmaster to navigate and use during a quiz session.
